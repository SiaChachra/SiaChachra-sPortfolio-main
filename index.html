Project 5

Java API Web Application

This Java-based project is a web application designed to search and display COVID-19 cases worldwide. Developed using SceneBuilder, it leverages an open API to fetch real-time data on COVID-19 statistics for various countries. The intuitive interface allows users to enter a country name and retrieve detailed information on COVID-19 cases. This project showcases skills in Java programming, API integration, and user interface design with SceneBuilder.

Project 6

JS Music Artist Web Application

This JavaScript-based web application allows users to search for an artist and view their songs, leveraging the iTunes API. The interface is user-friendly and responsive, displaying search results dynamically. Each song in the results links directly to iTunes, enabling users to listen or purchase tracks seamlessly. The site showcases skills in API integration, JavaScript, and web design.
